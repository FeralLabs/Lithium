Hello there, Friend!
Since, you're here you are probably wondering what Lithium is. So here it goes: It is a responsive framework built by Feral Labs. Everything you see here was built using this framework and the design can be change in a matter of a few seconds.

What's "Responsive"?

See for yourself, if you are on a PC, try to resize the window and enjoy. Everything rearranges to create a mobile friendly website. Lithium does this by itself, from its core, there is no need for extra code to support mobile browsers.

The Grid

Lithium has a fully responsive grid, that by default is 12 blocks, but it can be used in any way with a few minor changes, we even have a 24-grid version if that's what you like.
